### 3. Flash-3 token
- Added `[flash-3]` to `FLASH_TOKENS` and `FLASH_OFFSETS` in `src/lib/grades.js`.
- Resolves to 3 grades below the athlete's max boulder/rope grade.
- Previous tokens: `[flash]` (=max), `[flash-1]` (1 below), `[flash-2]` (2 below). Now also `[flash-3]`.

### 4. Tagging system for exercises and named workouts
Full tagging system added. **Requires running SQL migration in Supabase** (see below).

**New DB tables:**
- `tags (id, gym_id, name)` — gym-scoped, unique per gym, names stored lowercase
- `exercise_tags (exercise_id, tag_id)` — junction table
- `workout_tags (workout_id, tag_id)` — junction table
- All three have RLS enabled with gym-scoped policies

**New hook:** `src/hooks/useTags.js`
- `useTags()` returns: `tags, exerciseTags, workoutTags, addTag, deleteTag, toggleExerciseTag, toggleWorkoutTag`
- `exerciseTags` and `workoutTags` are maps of `id → tag[]`
- Tag names are lowercased on save

**Settings page (`src/pages/Settings.jsx`):**
- New **Tags section** (above Exercises) — create/delete gym-wide tags
- Exercise rows now show assigned tag chips inline to the right of the name
- `+ tag` button opens an inline picker to toggle tags on/off per exercise

**Workouts page (`src/pages/Workouts.jsx`):**
- Tag filter bar above the Named Workouts list — click tags to filter (OR logic)
- Each named workout card shows its tags with a `+ tag` picker

**WorkoutForm (`src/components/workouts/WorkoutForm.jsx`):**
- "Filter exercises" chip bar above the sets — narrows exercise dropdowns by tag
- Accepts `tags` and `exerciseTags` as optional props (default to empty)

**Supabase SQL to run:**
```sql
create table if not exists tags (
  id     uuid primary key default gen_random_uuid(),
  gym_id uuid references gyms(id) on delete cascade not null,
  name   text not null,
  unique(gym_id, name)
);
alter table tags enable row level security;
create policy "gym_tags" on tags for all
  using     (gym_id = (select gym_id from coaches where id = auth.uid()))
  with check (gym_id = (select gym_id from coaches where id = auth.uid()));

create table if not exists exercise_tags (
  exercise_id uuid references exercises(id) on delete cascade not null,
  tag_id      uuid references tags(id)      on delete cascade not null,
  primary key (exercise_id, tag_id)
);
alter table exercise_tags enable row level security;
create policy "gym_exercise_tags" on exercise_tags for all
  using     (exercise_id in (select id from exercises where gym_id = (select gym_id from coaches where id = auth.uid())))
  with check (exercise_id in (select id from exercises where gym_id = (select gym_id from coaches where id = auth.uid())));

create table if not exists workout_tags (
  workout_id uuid references workouts(id) on delete cascade not null,
  tag_id     uuid references tags(id)     on delete cascade not null,
  primary key (workout_id, tag_id)
);
alter table workout_tags enable row level security;
create policy "gym_workout_tags" on workout_tags for all
  using     (tag_id in (select id from tags where gym_id = (select gym_id from coaches where id = auth.uid())))
  with check (tag_id in (select id from tags where gym_id = (select gym_id from coaches where id = auth.uid())));
```
